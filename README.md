# TT (DOS 打字练习) - Go 复刻版

一个在终端中运行的 TT 风格打字练习程序，包含课程选择、实时错误高亮、速度/正确率统计。

## 功能

- 课程菜单（方向键或数字选择）
- 逐字输入对比（正确/错误颜色区分）
- Backspace 回删
- 实时统计：用时、WPM、CPM、正确率
- 完成后可重练或返回菜单

## 运行

```bash
go mod tidy
go run .
```

## 操作说明

- 菜单：
  - `↑/↓` 选择课程
  - `1..9` 快速选择课程
  - `Enter` 开始
  - `Q` 退出
- 练习中：
  - 普通键输入
  - `Backspace` 删除
  - `ESC` 返回菜单
  - `Ctrl+C` 退出
- 完成后：
  - `R` 重练
  - `M` 回菜单
  - `Q` 退出

## 备注

- 程序会进入终端原始模式以实现实时按键读取。
- 建议在支持 ANSI 颜色的终端中使用。
